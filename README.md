# AI Job Board

## Overview

AI Job Board is a modern job search platform built using Next.js, TypeScript, and Tailwind CSS. It allows users to browse jobs, search opportunities, filter listings, and bookmark jobs through a clean and responsive interface.

## Features

- Responsive UI
- Dark/Light Theme
- Search Jobs
- Filter by Location
- Filter by Job Type
- Bookmark Jobs
- Job Details Page
- Responsive Design

## Tech Stack

- Next.js 16
- React
- TypeScript
- Tailwind CSS
- Lucide React
- Vercel

## Installation

```bash
git clone https://github.com/tharunnelikondi97269-bot/ai-job-board.git
cd ai-job-board
npm install
npm run dev
```

Open:

```
http://localhost:3000
```
